深圳市英唐智能控制股份有限公司

关于公司董事、高级管理人员股份减持计划实施完毕的公告

公司董事兼总经理钟勇斌先生、董事 甘礼清先生保证向本公司提供的信息内容真实 、准确、完整，没有虚假记载、误导性陈述或重大遗漏。

深圳市英唐智能控制股份有限公司（以下简称“公司”）于2020年1月21日披露了公司控股股东胡庆周先生、董事兼总经理钟勇斌先生、董事甘礼清先生自2020年2月19日至2020年8月19日期间拟减持本公司股份的减持计划（具体内容详见公司在创业板指定信息披露媒体巨潮资讯网上披露的《关于公司控股股东、董事、高级管理人员股份减持预披露的公告》，公告编号：2020-010）。其中，公司董事兼总经理钟勇斌先生计划自2020年2月19日至2020年8月19日期间以集中竞价交易、大宗交易的方式减持本公司股份，减持总数不超过3,767,542股（占本公司总股本比例0.35%）；公司董事甘礼清先生计划自2020年2月19日至2020年8月19日期间以集中竞价交易、大宗交易的方式减持本公司股份，减持总数不超过8,000,000股（占本公司总股本比例0.75%）。

公司于近日收到钟勇斌先生、甘礼清先生的《关于股份减持计划实施完毕的告知函》，截至本公告日，钟勇斌先生累计减持股份3,747,300股，剩余还可减持股份不超过20,242股；甘礼清先生累计减持股份7,500,000股，剩余还可减持股份不超过500,000股，上述两位董事剩余未减持股份在原减持计划期限内将不再减持，本次减持计划实施完毕。现将具体事项公告如下：
